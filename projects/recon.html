RECONPLANNER

Scoped recon and surface map

Before you assess a client's external surface, you need a plan and a declared scope. ReconPlanner reads a scope file, enumerates subdomain candidates with passive DNS lookups, runs a safe port scan against localhost, and renders the result as an attack-surface graph plus a markdown report.

WHAT IT DOES

Scopedeclared domains
Enumerationpassive DNS
Scanlocalhost only
OutputSVG graph + report
Dependenciesnone, stdlib

STACK

Python 3.11 stdlib socket http.server

RUN IT

# no API key, standard library only
python3 app.py
# edit scope.json first, then:
curl localhost:8001/api/graph
curl localhost:8001/api/report
REPOSITORY → zero keyslocal-only scan

SAFETY MODEL

Active scanning targets 127.0.0.1 only. DNS runs passive lookups for declared domains. It never brute-forces external ranges or scans hosts you do not control. Point it at systems you have permission to test.
