FIELDCONSOLE

Live fleet ops console

A WebSocket-driven console for managing remote site deployments. Each site in your fleet has a live status. Hit Deploy and watch it flip from deploying to healthy in real time, with a streaming log tail. The thing you leave open on the second monitor during a rollout.

WHAT IT DOES

Fleet8 seeded sites
TransportWebSocket, live
ActionsDeploy per site
Logstreaming tail
Statehealthy / down / deploying

STACK

Node Express ws vanilla UI

RUN IT

# no API key, no cloud
npm install
npm start
# open the console, then:
curl localhost:3000/api/sites
REPOSITORY → zero keysruns offline

FIELD ORIGIN

Stands in for the deploy consoles used across multi-site rollouts, where one screen has to show every location's state and let you act without a page refresh.
